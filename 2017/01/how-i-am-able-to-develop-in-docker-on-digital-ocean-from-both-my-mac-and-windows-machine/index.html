Deploying and Managing Docker Instances on Digital Ocean From Both a Mac and Windows Machine

Jan 16th 2017 (Comments)

I've been working on a project that is hosted using Docker containers up on Digital Ocean. In an earlier post I walk through how to setup deploy and manage different environments.

The work of creating, deploying and essentially managing all of my containers up on Digital Ocean worked great, but only from the computer I initiated this process with. For this example, we're going to walk through how I was able to not only build/deploy from my Mac like I originaly did, but what steps I took to allow me to switch over to a Windows machine and do the exact same thing.

It's not obvious how to do it, and if you do some googling you'll most certanly land

  1. copy/zip folder on Mac
  2. unzip in same directory on Windows
  3. when I tried to run the docker-machine env command it failed because of the pem file not found - I had to manually create that directory on windows and copy the cert files in...
  4. Had to update the config.json and replace {{HOME}} with C:/Users/Jason

after looking at it I probably could have skipped step 3 above and just modified the config.json to point to the proper directories (instead of copying files around).
